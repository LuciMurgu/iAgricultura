## Number Formatting

- Decimal separator: comma (,) → "1.234,56"
- Thousands separator: period (.) → "34.250"
- Currency: value + space + symbol → "7.740 RON"
- Percentage: value + % → "97%"
- Date: DD.MM.YYYY → "28.03.2026" (or relative: "acum 2h")
